About skiftOS

skiftOS is an x86 operating system built from scratch in modern C++. It features a kernel named Hjert, a graphical user interface with a compositing window manager, and familiar UNIX utilities.


Features

  • Pre-emptive multitasking
  • A strong emphasis on user experience.
  • A compositing window manager (Compositor)
  • Has a wide range of applications such as an Image Viewer, Text Editor, and Terminal Emulator
  • It can run DOOM
